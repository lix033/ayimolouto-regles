# ayimolouto-regles
Bienvenue sur Ayimolouto. La protection de vos données personnelles est une priorité pour nous. Cette politique de confidentialité vous explique quelles informations nous collectons, comment nous les utilisons et quels sont vos droits lorsque vous utilisez notre application mobile.

En utilisant Ayimolouto, vous acceptez les pratiques décrites dans cette politique.
1. Les données que nous collectons

Pour vous aider à trouver le meilleur Ayimolou près de chez vous, nous pouvons collecter les informations suivantes :

    Données de localisation : Avec votre autorisation, nous collectons la position GPS exacte ou approximative de votre appareil. Cela nous permet de vous afficher les vendeurs d'Ayimolou les plus proches de vous sur la carte.

    Données d'utilisation : Nous recueillons des informations automatiques sur la façon dont vous interagissez avec l'application (type d'appareil utilisé, pages visitées, temps passé sur l'application, etc.) pour corriger les bugs et améliorer nos services.

2. Comment nous utilisons vos données

Vos données sont utilisées uniquement dans le but d'améliorer votre expérience. Nous les utilisons pour :

    Vous géolocaliser et vous indiquer le trajet vers les vendeurs d'Ayimolou.

    Vous permettre de publier des avis et des notes sur les vendeurs.

    Analyser l'utilisation de l'application pour en améliorer les performances et le design.

    Répondre à vos questions ou demandes d'assistance.

3. Partage de vos informations

Nous ne vendons, ne louons, ni ne partageons vos données personnelles à des tiers à des fins commerciales. Nous pouvons toutefois partager certaines données avec des prestataires de services de confiance qui nous aident à faire fonctionner l'application (par exemple, les services de cartographie comme Google Maps ou des outils d'analyse d'audience), sous stricte obligation de confidentialité.

4. Sécurité de vos données
Nous mettons en œuvre des mesures de sécurité techniques et organisationnelles (comme le cryptage des mots de passe) pour protéger vos informations personnelles contre tout accès, modification, divulgation ou destruction non autorisés. Toutefois, rappelez-vous qu'aucune méthode de transmission sur Internet ou de stockage électronique n'est sûre à 100 %.

6. Vos droits (Conformément à la législation togolaise)

Vous avez le contrôle sur vos données personnelles. Vous pouvez à tout moment :

    Désactiver la géolocalisation directement dans les paramètres de votre téléphone. L'application fonctionnera toujours, mais vous devrez chercher les lieux manuellement.

    Accéder, modifier ou supprimer les informations de votre profil depuis l'application.

    Demander la suppression complète de votre compte et de vos données en nous contactant directement.

6. Modifications de cette politique

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre pour refléter les changements apportés à notre application ou aux lois en vigueur. Nous vous informerons de toute modification importante via une notification dans l'application.
7. Nous contacter

Si vous avez des questions concernant cette politique de confidentialité ou la gestion de vos données, n'hésitez pas à nous contacter :
